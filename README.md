# TaskFlow Fullstack App

A modern full-stack Task Management application built using Django REST Framework and React.

## Features

- User Registration & Login
- JWT Authentication
- Create, Update & Delete Tasks
- Protected Routes
- Responsive UI
- REST API Integration
- Task Completion Tracking

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- CSS

### Backend
- Django
- Django REST Framework
- Simple JWT
- SQLite
